## ‚úÖ Global MarceloDesignX Styling Rules

| Element            | Rule                                                                 |
|--------------------|----------------------------------------------------------------------|
| Background         | Use `#121212` or `#1A1A1A` as base                                   |
| Font               | Use modern sans-serif (Inter, Poppins, or Satoshi)                   |
| Primary Accent     | `#F39C1F` (Orange ‚Äì Invite, Call-to-action, Live)                    |

| Border Radius      | 18px‚Äì24px for cards, modals, and inputs                             |
| Button Hover       | `transform: scale(1.05)` + soft shadow                              |
| Icon Animation     | Subtle bounce or glow on hover                                      |
| Depth Layers       | Floating containers with blur and shadows                          |
| Tooltip            | Show on hover (e.g. ‚ÄúClick to connect‚Äù, ‚ÄúUpgrade for full access‚Äù) |
